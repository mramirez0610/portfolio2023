# [Jungle Cook](https://in-info-web4.informatics.iupui.edu/~ramimarc/n315/jungleCook/dist/#home)

Project consisted of recreating a prototype and keeping design as accurate and consistent as possible. The prototype was provided by the client and the project was developed using JQuery and Sass. Firebase was used for user authentication, and Firestore for data storage. 

## Technology Used

- JQuery
- Firebase Authentication
- LocalStorage
