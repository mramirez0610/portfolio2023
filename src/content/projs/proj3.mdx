# What Should I Eat?

## Overview
___
What Should I Eat is a full-stack web application that I made in my free time.  I always find myself wanting to learn how to cook, though whenever I try to decide, I experience choice paralysis. Most of the time I just don't know what to make, I have all of these ingredients, but no ideas.

I use OpenAI's API to call a request, which contains the user's ingredients, as well as how much they like that item. The API call then returns a new recipe everytime according to ingredients and preference. It makes for a fun time when you're bored, and and even better time when you're hungry.

RBAC authorization techniques, as the user is required to either login, or login anonymously to use the service.
